# Test and Metrics
This directory contains scripts and metric implementations used to test, evaluate, and benchmark the LLM agent that generates Kubernetes manifests. The tools here compare generated manifests to reference manifests and to natural-language intent descriptions, producing quantitative scores and diagnostic outputs to guide agent improvements.

## Content

- `intent_generator_script.py`: Utility to create or extract natural-language intent descriptions from example manifests or model outputs for use as evaluation inputs.

- `intent_similarity.py`: Computes similarity between intents generated by different LLMs through embedding‑based and token‑based methods, allowing you to quantify how much rephrased intents diverge.

- `manifest_similarity.py`: High-level orchestration script that runs the available metrics over a set of predictions and references and aggregates results.

- `metrics/`
Directory containing all metrics implementations:
    - **Text-level metrics**: BLEU, Edit Distance and Exact Match
    - **YAML-Aware metrics**: Key-Value Exact Match, K-Value Wildcard Match and Label Match
